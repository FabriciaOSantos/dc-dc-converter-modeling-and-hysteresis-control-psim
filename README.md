# DC-DC Converter Modeling, PID and Hysteresis Control - PSIM

This repository contains an academic project for the Modeling and Control of Static Converters course, focused on the mathematical modeling, simulation and control of Buck and Boost DC-DC converters.

The project includes converter design, transfer function analysis, PSIM simulations and the implementation of PID and hysteresis control strategies.

## Objective

* Design Buck and Boost DC-DC converters
* Develop the mathematical modeling of the converters
* Simulate the circuits using PSIM
* Apply PID control to regulate the converter output
* Apply hysteresis control and analyze its behavior
* Compare the simulated responses of the converters

## Project description

The work presents the design and analysis of Buck and Boost converters using classical power electronics methods. The converters are modeled mathematically and simulated in PSIM to evaluate their dynamic behavior.

For the Buck converter, the project includes component sizing, transfer function analysis and closed-loop control simulation.

For the Boost converter, the project includes voltage gain analysis, component sizing, transfer function modeling and output voltage regulation using control strategies.

The project also explores hysteresis control as an alternative approach for converter regulation, analyzing the converter response under switching control conditions.
